# 아이노바

## 개요

아이노바는 Slack 채널 `#aiagent_기획-개발`에서 기획, 개발, 운영 이슈가 함께 논의되는 AI 에이전트 서비스입니다. 채널에서는 Jira 요약, 모델 추가/비활성화, 공지사항, 계정 운영, 검증 서버, 아티팩트/폴더 공유 기능, QA 흐름이 주로 다뤄지고 있습니다.

최근 대화에서 확인된 업무 축은 다음과 같습니다.

| 영역 | 주요 내용 |
| --- | --- |
| 모델 관리 | 신규 LLM 모델 추가, 모델 카테고리 조정, OpenAI 모델 비활성화 |
| 공지사항 | 공지사항 페이지, 읽음/Dot 표시 정책, 공지 등록 운영 |
| 계정 운영 | 휴직자 계정 비활성화, 관리자 페이지 토글 OFF 운영 |
| 검증/배포 | DEV, STG, 상용 반영 순서와 QA 방식 |
| 아티팩트 | 아티팩트 노출 속도 개선, 공유/게시 기능, 모델 비용 고려 |
| 폴더 공유 | 개인/팀 공유 범위, 읽기 전용 대화, 실시간 업데이트 알림 |
| 운영 분석 | LLM 사용량 분석, Input 중심 사용 패턴 모니터링 |
