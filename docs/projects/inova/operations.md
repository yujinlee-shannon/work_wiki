# 아이노바 운영 메모

## 공지사항 등록

공지사항은 기능 반영, 가이드 안내, 모델 업데이트, 장애/운영 변경이 있을 때 등록됩니다.

최근 공지 예시는 다음과 같습니다.

- Chrome 확장 프로그램 수동 설치 안내.
- 답변 중단 또는 불러오기 실패 시 가이드.
- 신규 LLM 모델 업데이트 안내.
- OpenAI 모델 비활성화와 모델 교체 관련 안내 예정.

## 휴직자 계정 처리

- 인사팀에서 매월 휴직자 명단을 전달받습니다.
- 관리자 페이지에서 휴직자 계정을 토글 OFF 처리합니다.
- 별도 개발 없이 기존 로그인 제한 메시지를 사용합니다.

## 모델 운영

- 신규 모델 추가 시 소개글과 카테고리를 함께 정리합니다.
- 사고 과정 노출이 필요한 기능에는 실제 추론 모델 여부를 확인합니다.
- 특정 provider 키 차단이나 정책 이슈가 발생하면 해당 provider 모델을 우선 비활성화하고 대체 모델을 적용합니다.

## QA와 배포 흐름

아이노바는 상황에 따라 DEV, 검증, 상용을 순차적으로 사용합니다.

- 수정사항이 많거나 빠른 피드백이 필요한 초기 QA는 DEV에서 확인합니다.
- DEV 확인 후 검증 서버에 반영합니다.
- 검증 확인 후 상용 반영을 요청합니다.
- 상용 반영 후에도 개선사항이 나오면 DEV부터 다시 확인해 순차 반영합니다.

## Jira/봇 요약

`aibizsolution-bot`이 CA Jira 요약과 IAIAO 업데이트 이슈 요약을 주기적으로 공유합니다.

현재 위키에는 봇 메시지 원문을 모두 복제하지 않고, 주요 기준일과 의미 있는 변화만 타임라인에 기록합니다.
