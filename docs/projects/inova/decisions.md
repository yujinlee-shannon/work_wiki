# 아이노바 결정 사항

## OpenAI 모델 비활성화와 title 요약 모델 교체

- 아이노바에서 사용 중인 OpenAI Key가 차단된 것으로 공유되었습니다.
- 우선 OpenAI 모델을 비활성화했습니다.
- title 요약에서 GPT 모델을 사용하던 부분도 Gemini 모델로 교체 후 배포하는 방향입니다.
- 배포 후 아이노바 공지사항 등록이 필요합니다.

## 신규 LLM 모델 카테고리 조정

- 신규 모델 추가 후 일부 모델에서 사고 과정 노출 방식이 다르게 보이는 이슈가 확인되었습니다.
- 추론 모델은 사고 과정 노출 프롬프트를 무시하도록 학습된 것으로 추정되었습니다.
- 사고 과정 노출이 필요한 분석 카테고리에는 실제 추론 모델만 넣는 방향을 고려하기로 했습니다.
- 해당 논의 이후 일부 모델은 idea 카테고리로 이동되었습니다.

## 휴직자 계정 운영

- 휴직자 계정 제한은 별도 개발 없이 수동으로 운영합니다.
- 인사팀에서 매월 휴직자 명단을 전달받고, 관리자 페이지에서 해당 인원의 계정을 토글 OFF 처리합니다.
- 토글 OFF 계정으로 로그인하면 프론트에서 관리자 문의 안내 메시지가 노출되므로 추가 개발은 필요하지 않은 것으로 정리되었습니다.

원문: <https://ai-tsv1393.slack.com/archives/C099HAM7U3C/p1775712723894389>

## 공지사항 Dot 표시 기준

- 공지사항의 Dot은 단순 신규 등록 여부가 아니라 읽지 않은 글 기준으로 표시합니다.
- 등록일 14일 이내이면서 미열람인 글에만 Dot을 표시합니다.
- LNB에는 조건에 해당하는 글이 1개 이상 있으면 Dot을 표시합니다.
- 리스트에는 해당 글 각각에 Dot을 표시합니다.
- 기존 공지사항 화면 진입 시 전체 읽음 처리하던 `mark-as-read` API는 필요 없어지는 방향입니다.
- 공지 상세의 이전글/다음글 navigation 정보에도 `isRead` 값이 필요합니다.

원문: <https://ai-tsv1393.slack.com/archives/C099HAM7U3C/p1771823326375179>

## 공지사항 이전글/다음글 배치

- 공지사항 상세 페이지의 이전글/다음글은 사용자가 순서대로 탐색하기 쉽도록 재검토가 필요했습니다.
- 원 논의는 상단 다음글, 하단 이전글 구조가 사용자에게 혼동을 줄 수 있다는 문제 제기에서 시작되었습니다.

## 폴더 공유 기능

- 폴더 메뉴에 폴더 공유하기 기능을 추가합니다.
- 모달에서 사용자 이름/이메일 검색 후 초대를 전송합니다.
- 이미 초대된 멤버 목록 확인과 삭제가 가능합니다.
- LNB에 공유받은 폴더 섹션을 추가하고, 기존 폴더는 내 폴더 섹션으로 분리합니다.
- 공유받은 폴더에서는 폴더 수정/삭제/공유 메뉴를 제한합니다.
- 공유받은 폴더의 대화는 읽기 전용이며, 채팅 입력창과 모델 선택기는 노출하지 않습니다.
- 생성자가 새 메시지를 보내면 상단에 새 대화 추가 배너를 노출하고, 새로고침 버튼으로 메시지를 로드합니다.
- 생성자가 공유 폴더를 삭제하면 초대된 사용자도 더 이상 접근할 수 없습니다.

원문: <https://ai-tsv1393.slack.com/archives/C099HAM7U3C/p1772088699032139>

## 폴더 공유 QA 흐름

- 초기 QA는 DEV 서버에서 빠르게 확인하고, 이후 검증 서버로 올리는 방식이 논의되었습니다.
- DEV 확인 후 검증 반영, 검증 확인 후 상용 반영으로 진행되었습니다.
- 상용 반영 후에도 모델 변경 시 대화 생성 위치와 초대 사용자 삭제 권한 개선사항이 추가로 확인되어 DEV, 검증, 상용에 순차 반영되었습니다.

## 아티팩트 노출 속도 개선

- 기존에는 아티팩트 영역과 대화 영역 구분을 위해 LLM이 2회 호출되어 지연이 발생했습니다.
- 아티팩트 에이전트 모델을 더 추론 성능이 좋은 모델로 지정하면 LLM 1회 호출로도 아티팩트와 대화 영역 구분이 가능하다고 판단했습니다.
- 비용을 고려해 고비용 모델 대신 더 비용 효율적인 Claude Sonnet 계열 모델 배포가 논의되었습니다.
- 검증 후 상용 MR이 반영되었고 동작 확인까지 완료되었습니다.
