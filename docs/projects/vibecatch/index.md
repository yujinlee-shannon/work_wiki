# 바이브캐치

## 개요

바이브캐치는 Slack 채널 `#바이브캐치`에서 화면설계, API 명세, Jira 진행 현황, 정책 확인, QA/버그 확인이 함께 논의되고 있는 프로젝트입니다.

최근 대화에서 주로 확인된 업무 축은 다음과 같습니다.

| 영역 | 주요 내용 |
| --- | --- |
| 화면설계 | v1.64~v1.68 화면설계서 업데이트, 알림 영역, 기기 관리, 테스트 시나리오 |
| 데이터/분석 | 여론 동향, AI 인사이트, 다음 주 예상 언급량, 동기간 대비 증감률 |
| SNS 연동 | 인스타그램 연동, 수집 중단 알림, 계정 등록자/관리자 권한 |
| 인증/기기 관리 | 기기 한도 초과, 재로그인, `pendingToken`, 신규 continue API |
| 리포트 | 키워드별 일간/주간/월간 요약 리포트, 8월 이후 기능 추가 |
| 운영/컴플라이언스 | ISMS-P 조치, 개인정보처리방침 링크 강조 |
