# 바이브캐치 운영 메모

## 화면설계서 업데이트 공지 패턴

화면설계서 업데이트는 버전, 업데이트 범위, 참고사항을 함께 공지합니다.

반복 확인된 참고사항:

- 페이지 새로고침 후에도 변경사항이 보이지 않으면 캐시 삭제가 필요할 수 있습니다.
- 상세 변경 내용은 Slack 스레드 또는 화면설계서 description을 확인합니다.

## Jira 요약

`aibizsolution-bot`이 Jira 요약 미리보기를 주기적으로 공유합니다.

요약에는 다음 정보가 포함됩니다.

- 기준일
- 총 건수
- 상태별 건수
- 댓글 있는 이슈와 없는 이슈의 정리 기준

현재 위키에는 Jira 요약 원문 전체를 복제하지 않고, 타임라인에서 주요 기준일과 상태 변화만 요약합니다.

## Slack 원문 보존 기준

- 결정이 난 스레드는 결정 사항 문서에 링크합니다.
- 단순 확인 요청은 이슈 문서에 한 줄로 남깁니다.
- 첨부 이미지/파일은 필요 시 제목과 파일 목적만 기록합니다.
- `.pLDH` 파일은 보안 프로그램 개입 가능성이 있으므로 원본 자료 확인 전 위키에 세부 내용을 전사하지 않습니다.
