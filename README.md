# 업무 위키

Slack 대화에서 나온 프로젝트 맥락, 결정 사항, 이슈, 운영 기준을 정리하는 Markdown 기반 위키입니다.

## 로컬 확인

Python과 MkDocs Material이 설치되어 있다면 아래 명령으로 확인할 수 있습니다.

```powershell
python -m pip install mkdocs-material
python -m mkdocs serve
```

브라우저에서 `http://127.0.0.1:8000`을 열면 됩니다.

## 정리 원칙

- Slack 원문 전체를 복사하지 않고, 업무에 남길 결론과 배경만 기록합니다.
- 민감정보, 계정정보, 비밀번호, 내부망 주소, 개인정보는 위키에 남기지 않습니다.
- 각 문서에는 가능한 경우 Slack 원문 링크를 남겨 추적 가능성을 유지합니다.
- 결정 사항과 미해결 이슈는 별도 문서에 분리해 나중에 검색하기 쉽게 유지합니다.
